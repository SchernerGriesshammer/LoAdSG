Detailed Description

Example:

 ShiftOperator Left(0,LeftShift);
 VectorSparseG a(AdaptiveSparseGrid);
 VectorSparseG b(AdaptiveSparseGrid);
 ...
 a = L(b);

Then a contains the hierarchical to the left shifted values of b.
